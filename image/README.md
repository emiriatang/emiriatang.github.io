# 图片文件夹说明

请将你的图片文件放入此文件夹，并按照以下命名规范：

## 必需图片

### 1. avatar.png (头像)
- **用途**: 个人主页头像
- **建议尺寸**: 200x200 像素或更大（正方形）
- **格式**: PNG（支持透明背景）或 JPG
- **说明**: 建议使用动漫风格的头像，效果更佳

## 可选图片（用于未来扩展）

### 2. banner.jpg (横幅背景)
- **用途**: 页面顶部横幅背景
- **建议尺寸**: 1920x400 像素
- **格式**: JPG

### 3. illustration-1.png, illustration-2.png (装饰插图)
- **用途**: 页面装饰元素
- **建议尺寸**: 根据设计需求
- **格式**: PNG（支持透明背景）

## 当前状态

- [x] avatar.png - 需要替换为你的头像
- [ ] banner.jpg - 可选
- [ ] 其他装饰图片 - 可选

## 注意事项

1. 如果你没有替换 avatar.png，页面会自动显示一个粉色的占位头像
2. 建议使用高质量图片以获得最佳视觉效果
3. 图片文件不要过大，建议控制在 500KB 以内以保证加载速度
4. 确保图片命名与代码中引用的一致

## 如何替换图片

1. 准备你的头像图片
2. 将图片重命名为 `avatar.png`
3. 将图片复制到本文件夹（image/）
4. 刷新网页即可看到效果

## 推荐图片资源

如果你需要找一些二次元风格的图片，可以考虑：
- 你喜欢的动漫角色官方图
- 自己绘制的原创角色
- 从合法渠道获取的免费素材
